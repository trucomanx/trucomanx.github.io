Programs - Generic:
Abc Gui Editor
GUI for working with abc notation. The abc-gui-editor program is a graphical user interface for working with abc notation, it was written in C++ with the QT libraries.
List-Data Tools
Set of graphical user interfaces for searching and listing: files in a directory, regions or points in images. The result is saved in a plain text file.
Smart News Organizer
Gui to help organize news using AI, It was written in Python with QT libraries.

Programs - Write articles:
Graph Generator Assistant
Gui to help generate professional graphics using AI, It was written in Python with QT libraries.
Clipboard Text Correction
Gui to help enhance your scientific writing with ease using AI, It was written in Python with QT libraries.
Clipboard Text Translate
Gui to help translate text with ease, It was written in Python with QT libraries.

Stand by:
Investment Tools
The investment-tools project brings together a set of programs that help us make calculations about personal finances. Compound interest, financial independence, etc.
PivMake GUI
The pivmake-gui program is a graphical user interface (GUI), written in C++ and QT libraries, for tracking regions in images using the particle image velocimetry (PIV) technique.
PDS Agronomic Tools
Programs written in Java and C++ that measure leaf area in pixels from a photo. Several example programs can be obtained from the agronomic-tools development page.
