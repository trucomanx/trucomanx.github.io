Bibliotecas:
Bio-Speckle Laser Tool Library (BSLTL)
Biblioteca para el procesamiento de imágenes digitales de speckle laser. La biblioteca está escrita usando el lenguaje m usado en OCTAVE y MATLAB.
PDS++ Project (libpdsmm)
Bibliotecas escritas en lenguaje C++ para el Procesamiento Digital de Señales. Cada una de las mini bibliotecas de PDS++ puede ser usada de forma independiente o en conjunto.
PDS Project (pdsplibrary)
Bibliotecas escritas en lenguaje C para el Procesamiento Digital de Señales. Cada una de las mini bibliotecas de PDS puede ser usada de forma independiente o en conjunto.

Programas:
Abc Gui Editor
GUI para trabajar con notación abc. El programa abc-gui-editor es una interfaz gráfica de usuario para trabajar con notación abc, fue escrito en C ++ con las librerías QT.
Investment Tools
El proyecto investment-tools agrupa un conjunto de programas que nos ajudan a fazer cálculos sobre finanças pessoais. Interes compuesto, independencia finaciera, etc.
PivMake GUI
El programa pivmake-gui es una interfaz gráfica de usuario para rastrear regiones en imágenes utilizando la técnica de velocimetría de imágenes de partículas (PIV).
PDS Agronomic Tools
Programas escritos en Java y C++ que a partir de una foto miden el area foliar en pixels. Desde la página de desarrollo de agronomic-tools se puede obtener varios programas de ejemplo.
List-Data Tools
Conjunto de interfaces gráficas de usuario para buscar y listar: archivos en un directorio, regiones o puntos en imágenes. El resultado se guarda en un archivo de texto plano.

Libros:
Métodos numéricos: Problemas não lineares e inversos
Este livro é uma coletânea de procedimentos relativos à solução de alguns problemas não lineares e inversos que têm aparecido, de forma muito recorrente, na minha vida profissional.
A practical guide to biospeckle laser analysis: theory and software
Libro sobre la biblioteca BSLTL para el procesamiento de imágenes digitales de speckle laser. La biblioteca está escrita usando el lenguaje m usado en OCTAVE y MATLAB.
Ingenharias, ciência e tecnologia 4 (Capítulo)
Participación con un artículo en el capítulo 27 titulado: Utilização da técnica de velocimetria por imagens de partículas (piv) para o estudo do módulo de elasticidade de painéis de madeira compensada.
Aulicha: crônicas nos Andes
Nesta obra o leitor conhecerá um conjunto de histórias acontecidas na Cordilheira dos Andes do Peru. Elas mostram algumas vivências do meu pai, Aurelio (1956), quando era apenas uma criança.
