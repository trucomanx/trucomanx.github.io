Bibliotecas:
Bio-Speckle Laser Tool Library (BSLTL)
Biblioteca para el procesamiento de imágenes digitales de speckle laser. La biblioteca está escrita usando el lenguaje m usado en OCTAVE y MATLAB.
PDS++ Project (libpdsmm)
Bibliotecas escritas en lenguaje C++ para el Procesamiento Digital de Señales. Cada una de las mini bibliotecas de PDS++ puede ser usada de forma independiente o en conjunto.
PDS Project (pdsplibrary)
Bibliotecas escritas en lenguaje C para el Procesamiento Digital de Señales. Cada una de las mini bibliotecas de PDS puede ser usada de forma independiente o en conjunto.
