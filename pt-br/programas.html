Programas - genericos:
Abc Gui Editor
GUI para trabajar con notación abc. El programa abc-gui-editor es una interfaz gráfica de usuario para trabajar con notación abc. Fue escrito en C ++ con las librerías QT.
List-Data Tools
Conjunto de interfaces gráficas de usuario para buscar y listar: archivos en un directorio, regiones o puntos en imágenes. El resultado se guarda en un archivo de texto plano.
Smart News Organizer
GUI para ajudar a organizar notícias usando a IA. Foi escrito em Python com bibliotecas QT.
Multiple Desktop Clocks
GUI para exibir vários fusos horários como relógios de escritorio. Foi escrito em Python com bibliotecas QT.
Simple Kanban GUI
GUI com interface de usuário gráfica kanban simples. Foi escrito em Python com bibliotecas QT.
Simple OcrMyPdf
GUI para OCRMYPDF: Apliqua OCR aos arquivos PDF sem texto. Foi escrito em Python com bibliotecas QT.
Graphviz Code Viewer
GUI para editar e visualizar o código GraphViz. Foi escrito em Python com bibliotecas QT.

Stand by:
Investment Tools
El proyecto investment-tools agrupa un conjunto de programas que nos ajudan a fazer cálculos sobre finanças pessoais. Interes compuesto, independencia finaciera, etc.
PivMake GUI
El programa pivmake-gui es una interfaz gráfica de usuario para rastrear regiones en imágenes utilizando la técnica de velocimetría de imágenes de partículas (PIV).
PDS Agronomic Tools
Programas escritos en Java y C++ que a partir de una foto miden el area foliar en pixels. Desde la página de desarrollo de agronomic-tools se puede obtener varios programas de ejemplo.
